【对话过程】
============================================================

[用户需求 09:00]
增加详细地址的录入，并对该地址调用高德API嵌入可以使其可以一键进行导航规划，
弹出窗口显示距离目的地多远，预计多少公里，在哪个方向，上不上高速，红绿灯有
多少个，预计驾驶时间。

[需求分析 09:01]
1. 录入保单页面需要新增"详细地址"区块，支持：
   - 省 / 市 / 区 三段式输入
   - 街道门牌号详细地址输入
   - 一键获取当前定位（逆地理编码回填）
   - 一键解析地址为经纬度（地理编码）
   - 经纬度实时展示
2. 保单列表页客户卡片新增"导航"按钮
3. 点击"导航"按钮触发以下流程：
   - 获取当前设备 GPS 位置
   - 调用高德驾车路径规划 v5 接口
   - 弹出 2027 科技风弹窗，展示：
     * 总距离（公里，1位小数）
     * 预计驾驶时间（分钟）
     * 目的地所在方向（8方位）
     * 是否上高速
     * 红绿灯数量
     * 过路费金额
     * 路线策略描述
4. 弹窗内提供"开始导航"按钮，跳转系统地图

[设计决策 09:02]
- 高德 API Key：7cf8dcc0a6d9547f616dbfc90e9463b5
- 调用接口：
  * 地理编码：https://restapi.amap.com/v3/geocode/geo
  * 逆地理编码：https://restapi.amap.com/v3/geocode/regeo
  * 驾车路径规划 v5：https://restapi.amap.com/v5/direction/driving
    （使用 show_fields=cost,tmcs,traffic_light 获取红绿灯与收费信息）
- 方位算法：基于经纬度差计算 8 方位（正东/东北/正北/西北/正西/西南/正南/东南）
- UI 风格：2027 科技风，橙金渐变地址录入区 + 蓝紫渐变弹窗
- 弹窗动效：fadeIn + slideUp 双重动画
- 关键数据"距离/时间/方向"采用大字号 + 渐变文字高亮
- 历史数据兼容：customer.address 可能为空，已做空值保护

[代码实现 09:03 - 09:25]
1. 类型扩展 (src/types/index.ts)
   - 新增 AddressInfo 接口（province/city/district/detail/fullAddress/longitude/latitude）
   - 新增 RoutePlanResult 接口（包含 12 个字段）
   - Customer 接口新增 address 字段

2. 高德地图服务模块 (src/services/amap.ts)
   - geocodeAddress(): 地址 → 经纬度
   - reverseGeocode(): 经纬度 → 地址
   - calcDirection(): 8 方位算法
   - drivingRoute(): 驾车路径规划（v5）
   - getCurrentLocation(): 获取当前位置
   - openNavigation(): 调起系统地图导航

3. 录入保单页面 (src/pages/add/index.tsx + index.module.scss)
   - 新增 6 个地址状态变量
   - 编辑模式自动回填地址
   - 新增"详细地址"卡片：省/市/区输入 + 详细地址 + 经纬度展示 + 三个操作按钮
   - 卡片采用橙金渐变 + 径向光晕装饰
   - saveCustomer 中构建 AddressInfo 对象
   - clearForm 中清空所有地址字段

4. 保单列表页 (src/pages/list/index.tsx + index.module.scss)
   - 新增 4 个导航状态：routePlanning/routeResult/routeModalOpen/routeTarget
   - 客户卡片新增"📍地址行"展示完整地址 + 解析状态
   - 客户卡片新增"🧭 导航"按钮（橙色渐变）
   - 新增 handleNavigate 方法：定位 → 规划 → 显示弹窗
   - 新增导航结果弹窗：
     * 顶部蓝紫渐变标题栏 + 圆形关闭按钮
     * 目的地信息卡片
     * 三栏数据展示区（距离/时间/方向），大字号 + 渐变文字
     * 详情列表（高速/红绿灯/过路费/策略/坐标）
     * 智能提示语（自动总结路线信息）
     * 操作按钮区（开始导航/关闭）
   - 错误状态友好展示

5. 样式细节
   - 所有新元素使用 2027 科技风配色（橙金/蓝紫渐变）
   - 弹窗采用毛玻璃遮罩 backdrop-filter
   - 数据卡片采用径向光晕装饰
   - 字体渐变使用 -webkit-background-clip: text
